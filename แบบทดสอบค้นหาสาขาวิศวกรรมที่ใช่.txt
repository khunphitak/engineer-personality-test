ส่วนที่ 1: แนวคิดและโครงสร้าง (อ้างอิงจาก 16 Personalities)
----------------------------------------------------------------
แบบทดสอบ 16 Personalities (อิงทฤษฎี MBTI / Big Five) มีโครงสร้างหลักคือ:

1) เป็น "ประโยคบอกเล่า" (statement) ไม่ใช่คำถามแบบมีถูก-ผิด
   ผู้ทำเพียงประเมินว่าตนเอง "เห็นด้วย" กับประโยคนั้นมากน้อยแค่ไหน

2) ใช้มาตรวัดแบบ Likert (เห็นด้วย <---> ไม่เห็นด้วย)
   ในที่นี้ใช้ 5 ระดับ เพื่อให้คะแนนง่ายและแม่นยำ

3) แต่ละข้อ "ผูก" กับมิติใดมิติหนึ่ง (เช่น เก็บตัว/เปิดเผย)
   เมื่อรวมคะแนนของแต่ละมิติ จะได้ "โปรไฟล์" ของผู้ทำ
   มิติที่ได้คะแนนสูงสุด = แนวโน้มเด่นของบุคคลนั้น

แบบทดสอบชุดนี้นำโครงสร้างเดียวกันมาปรับใช้ โดยเปลี่ยน "มิติบุคลิกภาพ"
เป็น "6 สาขาวิศวกรรม" แต่ละข้อสะท้อนความสนใจ/ความถนัดของสาขาหนึ่ง
เมื่อรวมคะแนนแล้ว สาขาที่ได้คะแนนสูงสุดคือสาขาที่เหมาะกับคุณที่สุด

6 สาขาที่วัด:
  C  = วิศวกรรมโยธา (Civil)
  M  = วิศวกรรมเครื่องกล (Mechanical)
  I  = วิศวกรรมอุตสาหการ (Industrial)
  Ch = วิศวกรรมเคมี (Chemical)
  E  = วิศวกรรมไฟฟ้า (Electrical)
  Co = วิศวกรรมคอมพิวเตอร์ (Computer)

ส่วนที่ 3: คำถาม (60 ข้อ)
----------------------------------------------------------------
แบ่งเป็น 2 กลุ่ม:
  3.1 ความสนใจทั่วไป (ข้อ 1-48)
  3.2 สถานการณ์ในชีวิตประจำวัน (ข้อ 49-60)

>>> 3.1 ความสนใจทั่วไป (ข้อ 1-48) <<<

ข้อที่ 1.  ฉันสนใจว่าตึกสูงและสะพานรับน้ำหนักและต้านแรงลม/แผ่นดินไหวได้อย่างไร   [ __ ]
ข้อที่ 2.  ฉันชอบแกะเครื่องจักรหรือเครื่องยนต์ออกมาดูว่ามันทำงานอย่างไร            [ __ ]
ข้อที่ 3.  ฉันชอบหาวิธีทำงานให้เร็วขึ้น ลดของเสีย และลดต้นทุน                      [ __ ]
ข้อที่ 4.  ฉันสนใจปฏิกิริยาเคมีและการเปลี่ยนวัตถุดิบให้กลายเป็นผลิตภัณฑ์            [ __ ]
ข้อที่ 5.  ฉันสนใจว่าระบบไฟฟ้าและการส่งจ่ายพลังงานทำงานอย่างไร                   [ __ ]
ข้อที่ 6.  ฉันชอบเขียนโปรแกรมและรู้สึกสนุกเวลาแก้บั๊กในโค้ด                        [ __ ]

ข้อที่ 7.  ฉันชอบออกไซต์ก่อสร้างและอยากเข้าใจขั้นตอนการสร้างอาคาร                [ __ ]
ข้อที่ 8.  ฉันสนใจเรื่องการเคลื่อนที่ กลไก เฟือง และระบบส่งกำลัง                    [ __ ]
ข้อที่ 9.  ฉันสนใจการจัดการสายการผลิตและกระบวนการต่าง ๆ ในโรงงาน               [ __ ]
ข้อที่ 10. ฉันชอบเรียนวิชาเคมีและสนุกกับการทดลองในห้องแล็บ                       [ __ ]
ข้อที่ 11. ฉันชอบต่อวงจรไฟฟ้า/อิเล็กทรอนิกส์และทดลองเล่นกับมัน                    [ __ ]
ข้อที่ 12. ฉันสนใจอัลกอริทึมและการแก้ปัญหาด้วยตรรกะอย่างเป็นขั้นตอน               [ __ ]

ข้อที่ 13. ฉันสนใจวัสดุก่อสร้าง เช่น คอนกรีต เหล็ก และคุณสมบัติการรับแรง           [ __ ]
ข้อที่ 14. ฉันสนใจเรื่องความร้อน พลังงาน และเครื่องยนต์ (อุณหพลศาสตร์)            [ __ ]
ข้อที่ 15. ฉันชอบวิเคราะห์ข้อมูลเพื่อปรับปรุงประสิทธิภาพการทำงาน                   [ __ ]
ข้อที่ 16. ฉันสนใจกระบวนการผลิตในโรงงาน เช่น ปิโตรเคมี อาหาร หรือยา             [ __ ]
ข้อที่ 17. ฉันสนใจเรื่องสัญญาณ คลื่น และระบบการสื่อสาร                           [ __ ]
ข้อที่ 18. ฉันชอบสร้างแอป เว็บไซต์ หรือซอฟต์แวร์ขึ้นมาเอง                          [ __ ]

ข้อที่ 19. ฉันชอบงานที่ได้ออกสำรวจพื้นที่จริงมากกว่านั่งในออฟฟิศตลอดเวลา           [ __ ]
ข้อที่ 20. ฉันชอบออกแบบชิ้นส่วนหรืออุปกรณ์ที่เคลื่อนไหวได้                          [ __ ]
ข้อที่ 21. ฉันสนใจระบบโลจิสติกส์ คลังสินค้า และห่วงโซ่อุปทาน                      [ __ ]
ข้อที่ 22. ฉันสนใจเรื่องการกลั่น การผสม และการแยกสาร                            [ __ ]
ข้อที่ 23. ฉันชอบทำงานกับไมโครคอนโทรลเลอร์และเซ็นเซอร์                          [ __ ]
ข้อที่ 24. ฉันสนใจปัญญาประดิษฐ์ (AI) และการวิเคราะห์ข้อมูลขนาดใหญ่               [ __ ]

ข้อที่ 25. ฉันสนใจการวางผังเมือง ระบบถนน และการจัดการ/ระบายน้ำ                  [ __ ]
ข้อที่ 26. ฉันสนใจหุ่นยนต์ ระบบอัตโนมัติเชิงกล และแขนกล                          [ __ ]
ข้อที่ 27. ฉันชอบวางแผน จัดตารางงาน และบริหารทรัพยากรให้คุ้มค่า                   [ __ ]
ข้อที่ 28. ฉันใส่ใจความปลอดภัยของสารเคมีและการควบคุมปฏิกิริยาไม่ให้เกิดอันตราย     [ __ ]
ข้อที่ 29. ฉันสนใจระบบควบคุมอัตโนมัติและอุปกรณ์อิเล็กทรอนิกส์                      [ __ ]
ข้อที่ 30. ฉันสนใจระบบเครือข่าย ฐานข้อมูล และความปลอดภัยไซเบอร์                  [ __ ]

ข้อที่ 31. ฉันภูมิใจกับงานที่จับต้องได้และคงอยู่ยาวนานเป็นสิบ ๆ ปี                   [ __ ]
ข้อที่ 32. ฉันชอบเขียนแบบ 3 มิติ (CAD) ของชิ้นส่วนเครื่องกล                       [ __ ]
ข้อที่ 33. ฉันสนใจเรื่องการควบคุมคุณภาพและการทำงานตามมาตรฐาน                   [ __ ]
ข้อที่ 34. ฉันสนใจการพัฒนาวัสดุใหม่ พอลิเมอร์ หรือพลังงานชีวภาพ                   [ __ ]
ข้อที่ 35. ฉันสนใจพลังงานหมุนเวียน เช่น โซลาร์เซลล์ และระบบไฟฟ้ากำลัง            [ __ ]
ข้อที่ 36. ฉันนั่งทำงานกับคอมพิวเตอร์ได้นาน ๆ เพื่อสร้างสิ่งใหม่โดยไม่เบื่อ           [ __ ]

ข้อที่ 37. ฉันชอบวิเคราะห์แรงในโครงสร้าง (คาน เสา ฐานราก) ว่าจะไม่พังได้อย่างไร   [ __ ]
ข้อที่ 38. ฉันสนใจยานยนต์ อากาศยาน หรือระบบขับเคลื่อนต่าง ๆ                     [ __ ]
ข้อที่ 39. ฉันชอบงานที่เชื่อมโยงทั้งคน เครื่องจักร และระบบเข้าด้วยกัน               [ __ ]
ข้อที่ 40. ฉันสนใจเรื่องสิ่งแวดล้อม การบำบัดของเสีย และกระบวนการที่ยั่งยืน          [ __ ]
ข้อที่ 41. ฉันชอบวิเคราะห์วงจรไฟฟ้าด้วยคณิตศาสตร์และฟิสิกส์                       [ __ ]
ข้อที่ 42. ฉันสนใจการออกแบบโครงสร้างซอฟต์แวร์และระบบขนาดใหญ่                   [ __ ]

ข้อที่ 43. ฉันสนใจเรื่องดิน ธรณีวิทยา และผลกระทบต่อการก่อสร้าง                    [ __ ]
ข้อที่ 44. ฉันชอบหาสาเหตุว่าทำไมเครื่องจักรสั่น ร้อนเกินไป หรือเสียหาย             [ __ ]
ข้อที่ 45. ฉันสนใจสถิติและการวิจัยดำเนินงาน (Optimization) เพื่อหาทางที่ดีที่สุด     [ __ ]
ข้อที่ 46. ฉันชอบคำนวณสมดุลมวลและพลังงานของกระบวนการผลิต                     [ __ ]
ข้อที่ 47. ฉันสนใจการออกแบบแผงวงจรและชิปอิเล็กทรอนิกส์                          [ __ ]
ข้อที่ 48. ฉันชอบเรียนรู้ภาษาโปรแกรมและเทคโนโลยีใหม่ ๆ อยู่เสมอ                  [ __ ]


>>> 3.2 สถานการณ์ในชีวิตประจำวัน (ข้อ 49-60) <<<
(สะท้อนพฤติกรรมจริงในชีวิตประจำวัน)

ข้อที่ 49. เวลาเดินทางผ่านสะพานหรือทางด่วน ฉันมักสังเกตโครงสร้างและสงสัยว่าสร้างอย่างไร  [ __ ]
ข้อที่ 50. เวลาของใช้ในบ้าน (พัดลม จักรยาน) เสีย ฉันมักลองแกะออกมาซ่อมเอง            [ __ ]
ข้อที่ 51. เวลาต่อคิวหรือร้านบริการช้า ฉันมักคิดว่าจัดระบบใหม่ให้เร็วขึ้นได้อย่างไร      [ __ ]
ข้อที่ 52. เวลาทำอาหารหรือผสมเครื่องดื่ม ฉันสนใจว่าอุณหภูมิ/สัดส่วนเปลี่ยนผลลัพธ์อย่างไร [ __ ]
ข้อที่ 53. เวลาอุปกรณ์ไฟฟ้าในบ้านมีปัญหา ฉันชอบหาสาเหตุเรื่องวงจร/ปลั๊ก/ฟิวส์เอง       [ __ ]
ข้อที่ 54. เวลาคอมหรือมือถือมีปัญหา ฉันชอบลองแก้ไขเองก่อนเรียกช่าง                  [ __ ]

ข้อที่ 55. เวลาฝนตกหนักน้ำท่วม ฉันคิดถึงว่าระบบระบายน้ำของเมืองควรปรับปรุงอย่างไร     [ __ ]
ข้อที่ 56. ฉันชอบสังเกตว่าเครื่องใช้ไฟฟ้า/รถยนต์ทำงานเงียบหรือสั่นผิดปกติหรือไม่        [ __ ]
ข้อที่ 57. ฉันชอบจัดบ้าน จัดโต๊ะ หรือวางแผนงานให้เป็นระเบียบและมีประสิทธิภาพ         [ __ ]
ข้อที่ 58. ฉันชอบอ่านส่วนผสมบนฉลากผลิตภัณฑ์และสงสัยว่าแต่ละอย่างทำหน้าที่อะไร        [ __ ]
ข้อที่ 59. ฉันสนใจว่าแบตเตอรี่มือถือและเครื่องชาร์จทำงานและเสื่อมสภาพอย่างไร          [ __ ]
ข้อที่ 60. ฉันชอบตั้งค่าแอปหรือเขียนสูตร/สคริปต์เล็ก ๆ เพื่อทำงานซ้ำ ๆ ให้อัตโนมัติ      [ __ ]

ส่วนที่ 4: วิธีรวมคะแนน (Scoring Key)
----------------------------------------------------------------
นำคะแนน (1-5) ของแต่ละข้อมาบวกกันตามกลุ่มสาขาด้านล่าง
แต่ละสาขามี 10 ข้อ คะแนนเต็ม = 50 คะแนน

  วิศวกรรมโยธา (Civil)
    = ข้อ 1 + 7 + 13 + 19 + 25 + 31 + 37 + 43 + 49 + 55 = ____ /50
  วิศวกรรมเครื่องกล (Mech.)
    = ข้อ 2 + 8 + 14 + 20 + 26 + 32 + 38 + 44 + 50 + 56 = ____ /50
  วิศวกรรมอุตสาหการ (Ind.)
    = ข้อ 3 + 9 + 15 + 21 + 27 + 33 + 39 + 45 + 51 + 57 = ____ /50
  วิศวกรรมเคมี (Chem.)
    = ข้อ 4 + 10 + 16 + 22 + 28 + 34 + 40 + 46 + 52 + 58 = ____ /50
  วิศวกรรมไฟฟ้า (Elec.)
    = ข้อ 5 + 11 + 17 + 23 + 29 + 35 + 41 + 47 + 53 + 59 = ____ /50
  วิศวกรรมคอมพิวเตอร์ (Comp.)
    = ข้อ 6 + 12 + 18 + 24 + 30 + 36 + 42 + 48 + 54 + 60 = ____ /50

คิดเป็นเปอร์เซ็นต์:  (คะแนนสาขา ÷ 50) × 100 = ____ %

การแปลผล:
  - สาขาที่ได้คะแนน "สูงสุด" คือสาขาที่เหมาะกับคุณมากที่สุด
  - หากมี 2-3 สาขาคะแนนใกล้กัน (ห่างไม่เกิน 3 คะแนน) แสดงว่าคุณมี
    ความสนใจหลากหลาย พิจารณาสาขาที่เป็นลูกผสมได้ (เช่น เมคคาทรอนิกส์)

ระดับความเข้มของความสนใจ (อ้างอิงจากเปอร์เซ็นต์ในสาขาที่สูงสุด):
  80-100%  = เหมาะมาก สนใจและถนัดชัดเจน
  60-79%   = เหมาะดี มีแนวโน้มไปทางนี้
  40-59%   = ปานกลาง ควรหาประสบการณ์เพิ่มเพื่อยืนยัน
  ต่ำกว่า 40% = ยังไม่ชัด ลองสำรวจหลายสาขาเพิ่มเติม

ส่วนที่ 5: คำอธิบายแต่ละสาขา (ผลการวิเคราะห์)
----------------------------------------------------------------

[ C ] วิศวกรรมโยธา (Civil Engineering)
จุดเด่นของคุณ: ชอบงานที่จับต้องได้ มองภาพใหญ่ ใส่ใจความปลอดภัยและความ
มั่นคงของสิ่งก่อสร้าง อดทนกับงานภาคสนาม
ทำอะไร: ออกแบบและควบคุมการก่อสร้างอาคาร สะพาน ถนน เขื่อน ระบบประปา
และโครงสร้างพื้นฐาน
เส้นทางอาชีพ: วิศวกรโครงสร้าง, วิศวกรสนาม, วิศวกรขนส่ง, ที่ปรึกษาก่อสร้าง,
งานราชการด้านโยธาธิการ

[ M ] วิศวกรรมเครื่องกล (Mechanical Engineering)
จุดเด่นของคุณ: ชอบเข้าใจกลไก สนใจการเคลื่อนที่ พลังงานและความร้อน ชอบ
ออกแบบและประดิษฐ์สิ่งที่ขยับได้
ทำอะไร: ออกแบบเครื่องจักร เครื่องยนต์ ระบบทำความเย็น หุ่นยนต์ ยานยนต์
และระบบส่งกำลัง
เส้นทางอาชีพ: วิศวกรออกแบบเครื่องกล, วิศวกรยานยนต์/อากาศยาน, วิศวกร
ซ่อมบำรุง, วิศวกรหุ่นยนต์/เมคคาทรอนิกส์

[ I ] วิศวกรรมอุตสาหการ (Industrial Engineering)
จุดเด่นของคุณ: ชอบมองภาพรวมของระบบ คิดเชิงประสิทธิภาพ ชอบวางแผนและ
จัดการ เชื่อมโยงคน-เครื่อง-ข้อมูลเข้าด้วยกัน
ทำอะไร: ปรับปรุงกระบวนการผลิต ลดต้นทุน บริหารคุณภาพ โลจิสติกส์ และ
ห่วงโซ่อุปทานให้มีประสิทธิภาพสูงสุด
เส้นทางอาชีพ: วิศวกรกระบวนการ, วิศวกรคุณภาพ, นักวางแผนการผลิต, ที่ปรึกษา
ปรับปรุงประสิทธิภาพ, ผู้จัดการโรงงาน

[ Ch ] วิศวกรรมเคมี (Chemical Engineering)
จุดเด่นของคุณ: ชอบเคมีและการทดลอง สนใจการเปลี่ยนวัตถุดิบเป็นผลิตภัณฑ์
ใส่ใจความปลอดภัยและสิ่งแวดล้อม คิดเชิงกระบวนการ
ทำอะไร: ออกแบบและควบคุมกระบวนการผลิตในอุตสาหกรรมปิโตรเคมี อาหาร ยา
พลังงาน และวัสดุ รวมถึงการบำบัดของเสีย
เส้นทางอาชีพ: วิศวกรกระบวนการเคมี, วิศวกรโรงงาน, วิศวกรความปลอดภัย,
นักวิจัยและพัฒนาผลิตภัณฑ์ (R&D), วิศวกรสิ่งแวดล้อม

[ E ] วิศวกรรมไฟฟ้า (Electrical Engineering)
จุดเด่นของคุณ: ชอบวงจรและอิเล็กทรอนิกส์ สนใจพลังงานไฟฟ้า สัญญาณ และ
ระบบควบคุม คิดเชิงคณิตศาสตร์/ฟิสิกส์ได้ดี
ทำอะไร: ออกแบบระบบไฟฟ้ากำลัง วงจรอิเล็กทรอนิกส์ ระบบสื่อสาร ระบบ
ควบคุมอัตโนมัติ และพลังงานหมุนเวียน
เส้นทางอาชีพ: วิศวกรไฟฟ้ากำลัง, วิศวกรอิเล็กทรอนิกส์, วิศวกรระบบควบคุม,
วิศวกรโทรคมนาคม, วิศวกรพลังงาน

[ Co ] วิศวกรรมคอมพิวเตอร์ (Computer Engineering)
จุดเด่นของคุณ: ชอบเขียนโค้ดและคิดเชิงตรรกะ สนใจอัลกอริทึม ข้อมูล และระบบ
เรียนรู้เทคโนโลยีใหม่ได้เร็ว นั่งจดจ่อกับงานได้นาน
ทำอะไร: พัฒนาซอฟต์แวร์ แอป เว็บไซต์ ระบบ AI ฐานข้อมูล เครือข่าย และ
ความปลอดภัยไซเบอร์
เส้นทางอาชีพ: นักพัฒนาซอฟต์แวร์, วิศวกรข้อมูล/AI, วิศวกรระบบ/เครือข่าย,
ผู้เชี่ยวชาญความปลอดภัยไซเบอร์, สถาปนิกซอฟต์แวร์
